# User Guide

## Features 
1. List all tasks
2. Add a todo task
3. Add an event task
4. Add a deadline task
5. Mark a task as done
6. Delete a task from the list
7. Update a task description and time
8. Find a task by keyword

## List all tasks
### Usage
### `list`

Displays an indexed list of all tasks.

Example of usage: 

`list`

Expected outcome:

`1. [D][X] Assignment (by: 20 Apr 2019)`
